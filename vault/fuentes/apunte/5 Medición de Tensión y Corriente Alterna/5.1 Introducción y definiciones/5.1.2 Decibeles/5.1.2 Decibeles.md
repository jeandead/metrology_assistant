## 5.1.2. Decibeles

A menudo, cuando se mide una variable, no se requiere conocer su valor absoluto, sino que se prefiere saber su valor relativo respecto a un valor de referencia. Por ejemplo, en un amplificador interesa conocer la relación de la potencia de salida respecto a la potencia de entrada, justamente para obtener la ganancia de amplificación: G = P out /P in .

Normalmente se prefiere el manejo de las ganancias en escala logarítmica base 10. Dado que una ganancia G es adimensional, la ganancia en escala logarítmica ó log 10 ( G ) , para ser diferenciada de la anterior se le asigna la unidad 'Belio' o 'Bel' 2 . Sin embargo, por su orden de magnitud se prefiere usar la décima parte, es decir, 'el decibel (dB)' .

Así la ganancia de potencia G en dB o simplemente G dB queda definido como:

$$G _ { d B } = 1 0 \cdot \log _ { 1 0 } \left ( \frac { P _ { o u t } } { P _ { i n } } \right ) \ \ [ d B ]$$

Las ventajas de usar el decibel son:

- a.Para obtener la ganancia total de un conjunto de redes o sistemas conectados en cascada, basta con sumar las ganancias en decibeles de cada una. Sin esta definición, habría que multiplicar las ganancias o funciones de transferencias.
- b.Si se quiere graficar una ganancia, que puede tener una gran variación, la escala logarítmica en general, permite observarla como si fuese lineal.

En electricidad cuando se tiene por objetivo el estudio redes y circuitos, la definición de decibel en términos de la potencia no es tan comúnmente usada y dado que resulta de mayor utilidad expresar la relaciones de ganancia o transferencia en términos de tensiones o corrientes, se llega a:

$$G _ { d B } = 1 0 \cdot \log _ { 1 0 } \left ( \frac { V _ { o u t } ^ { 2 } } { R _ { o u t } } \cdot \frac { R _ { i n } } { V _ { i n } ^ { 2 } } \right ) = 2 0 \cdot \log _ { 1 0 } \left ( \frac { V _ { o u t } } { V _ { i n } } \right ) \ \ [ d B ]$$

Luego, la definición anterior se usa directamente sin necesidad de verificar o asegurar la igualdad de las resistencias R in = R out .
