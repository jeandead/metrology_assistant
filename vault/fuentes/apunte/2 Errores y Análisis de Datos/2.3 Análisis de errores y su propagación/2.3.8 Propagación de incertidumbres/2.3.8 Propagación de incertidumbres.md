## 2.3.8. Propagación de incertidumbres

A partir de la función en varias variables f ( ⃗ x ) y las incertidumbres asociadas a cada una de las variables de entrada x i , llamadas µ x i , se obtiene a partir de la serie de Taylor , pero considerando la métrica Euclidiana o Pitagórica la siguiente propagación de incertidumbres :

$$\mu _ { y } = \sqrt { \sum _ { i = 1 } ^ { n } \left \{ \frac { \partial f } { \partial x _ { i } } \Big | _ { \overline { x } _ { j } = 1 \longrightarrow n } \cdot \mu _ { x _ { i } } \right \} ^ { 2 } } = \sqrt { \sum _ { i = 1 } ^ { n } \left \{ \mu _ { y _ { i } } \right \} ^ { 2 } } \\$$

Para finalizar y determinar la 'incertidumbre total' , hay que considerar la probabilidad de cobertura o intervalo de confianza o simplemente incertidumbre expandida , que se interpreta de cierto punto de vista como la forma en que los grados de libertad finales o efectivos interactúan entre sí. Para determinar los grados de libertad efectivos, se tiene:

$$\nu _ { e f } = \frac { \mu _ { y } ^ { 4 } } { \sum _ { i = 1 } ^ { n } \frac { \mu _ { y _ { i } } ^ { 4 } } { \nu _ { x _ { i } } } } & & ( 2 . 1 1 ) \\$$

Con lo cual la incertidumbre expandida queda definida como:

$$\mu _ { y k } = k \cdot \mu _ { y }$$

Donde k es el factor de cobertura, a partir de la probabilidad de cobertura ( p % ) que se desee para los grados de libertad efectivos que tenga el resultado del proceso ' y '.

La tabla 2.4 muestra los factores de cobertura en función de los grados de libertad efectivos 7 para una probabilidad de cobertura de 95 , 45 % (caso de ejemplo), valores que están basados en asumir que se tiene una distribución ' t '. La distribución ' t ' se utiliza cuando no se conoce la desviación de una muestra de datos típicamente menor a 30 datos, es similar a la Normal, pero con mayor área en los extremos y menos en el centro. Su uso típico es para establecer intervalos de confianza.

Ejemplo: Se desea calcular el volumen de una plancha metálica, para ello se disponen medidas del largo y ancho, mientras que el espesor es un dato del fabricante, teniendo el siguiente resumen de datos previamente procesados:

- Largo ( l ), cantidad de mediciones n l = 9 , valor medio ¯ l = 10 , 00cm , desviación estándar s l = 1 , 00cm . ⇒ Dato tipo A .
- Ancho ( a ), cantidad de mediciones n a = 16 , valor medio ¯ a = 5 , 00cm , desviación estándar s l = 0 , 50cm . ⇒ Dato tipo A .
- Espesor ( e ), dato del fabricante e = 0 , 010cm ± 2 % . ⇒ Dato tipo B .

Respuesta: Se define la función de salida 'volumen ( v )' o simplemente y = v , tal que:

$$y = l \cdot a \cdot e \ [ c m ^ { 3 } ]$$

El valor medio del volumen rápidamente se obtiene de:

$$\bar { y } \equiv \bar { v } = 1 0 , 0 0 \cdot 5 , 0 0 \cdot 0 , 0 1 = 0 , 5 0 0 = 5 0 0 \cdot 1 0 ^ { - 3 } \ [ c m ^ { 3 } ]$$

Para analizar la propagación de incertidumbres, se deben calcular o determinar en función de los datos disponibles según:

$$\nu _ { \ } \mu _ { l } = \frac { 1 , 0 0 } { \sqrt { 9 } } = 0 , 3 3 3 c m ; \nu _ { l } = 8$$

$$\mathcal { V } \ \mu _ { a } = \frac { 0 , 5 0 } { \sqrt { 1 6 } } = 0 , 1 3 c m ; \nu _ { a } = 1 5$$

$$\sqrt { \mu _ { e } } = \frac { 0 , 0 1 0 * 2 / 1 0 0 } { \sqrt { 3 } } = 1 1 5 \cdot 1 0 ^ { - 6 } c m ; \nu _ { e } \rightarrow \infty$$

$$& l = \frac { 1 , 0 0 } { \sqrt { 9 } } = 0 , 3 3 3 c m ; \, \nu _ { l } = 8 \\ & a = \frac { 0 , 5 0 } { \sqrt { 1 6 } } = 0 , 1 3 c m ; \, \nu _ { a } = 1 5 \\ & e = \frac { 0 , 0 1 0 * 2 / 1 0 0 } { \sqrt { 3 } } = 1 1 5 \cdot 1 0 ^ { - 6 } c m ; \, \nu _ { e } \to \infty$$

Por tanto la distribución de incertidumbres queda:

$$\mu _ { y } ^ { 2 } = \left ( \underbrace { \bar { a } \cdot \bar { e } \cdot \mu _ { l } } _ { 0 , 0 1 7 } \right ) ^ { 2 } + \left ( \underbrace { \bar { l } \cdot \bar { e } \cdot \mu _ { a } } _ { 0 , 0 1 3 } \right ) ^ { 2 } + \left ( \underbrace { \bar { l } \cdot \bar { a } \cdot \mu _ { e } } _ { 0 , 0 0 5 8 } \right ) ^ { 2 } \Longleftrightarrow \mu _ { y } = 0 , 0 2 2 \ [ c m ^ { 3 } ]$$
