# 5.1.4 Diagrama Logarítmico o de Bode

Tags: #eli214




El diagrama logarítmico o de diagrama Bode permite representar a una función compleja H ( ω ) , que típicamente es la respuesta en frecuencias de un sistema , por medio de dos gráficos en términos de la frecuencia ω , que va desde 0 hasta ∞ :

1. ‖ H ( ω ) ‖ v/s ω es conocido como Diagrama de Magnitud.
2. ∡ H ( ω ) v/s ω es conocido como Diagrama de Fase.

En general, H ( ω ) siempre puede escribirse como:

$$H ( \omega ) = \| H ( \omega ) \| \triangleleft ( H ( \omega ) = K _ { B } \cdot \left ( \frac { \prod _ { i } \left \{ \frac { j \omega } { z _ { i } } + 1 \right \} } { ( j \omega ) ^ { q } \prod _ { k } \left \{ \frac { j \omega } { p _ { k } } + 1 \right \} } \right ) \\$$

conocida como la Forma de Bode , donde K B es la ganancia de Bode y q es el número de polos ( q &gt; 0) o ceros ( q &lt; 0) en el origen ( ω = 0 ). Por otro lado, z i = ω c i son los ' i ' ceros de H ( ω ) que por construcción está descritos en términos de frecuencias de corte, mientras que p k = ω c k son los ' k ' polos de de H ( ω ) , también expresados en términos de frecuencias de corte.

Entonces se tiene para la magnitud, la no sencilla descripción:



$$ \| \mathbf{H}(\omega) \| = \| K_B \| \cdot \left( \frac{\prod_i \| \left\{ \frac{j\omega}{z_i} + 1 \right\} \|}{(\omega)^q \prod_k \| \left\{ \frac{j\omega}{p_i} + 1 \right\} \|} \right) = \| K_B \| \cdot \left( \frac{\prod_i \left\{ \sqrt{\left(\frac{\omega}{z_i}\right)^2 + 1} \right\}}{(\omega)^q \prod_k \left\{ \sqrt{\left(\frac{\omega}{p_i}\right)^2 + 1} \right\}} \right)  $$





y para la fase:

$$\triangle H ( \omega ) = \underbrace { \triangle K _ { B } } _ { 0 ^ { o } \vee 1 8 0 ^ { o } } + \sum _ { i } \left \{ t a n ^ { - 1 } \left ( \frac { \omega } { z _ { i } } \right ) \right \} - 9 0 ^ { o } q - \sum _ { k } \left \{ \tan ^ { - 1 } \left ( \frac { \omega } { p _ { k } } \right ) \right \}$$

Para que el diagrama de magnitud de la ecuación (5.3) pueda obtenerse por sumas, se debe graficar en decibeles ( dB ) , es decir, usar la propiedad del logaritmo que transforma los productos en sumas:

$$\| \mathbf H ( \omega ) \| _ { d B } = 2 0 l o g _ { 1 0 } \| \mathbf H ( \omega ) \| = 2 0 l o g \| \mathbf H ( \omega ) \|$$

De este modo se llega a una descripción de la magnitud como:

$$\| H ( \omega ) \| _ { d B } = \| K _ { B } \| _ { d B } - 2 0 q l o g ( \omega ) \pm \sum _ { l } \left \{ 1 0 l o g \left \{ \left ( \frac { \omega } { \omega _ { c _ { l } } } \right ) ^ { 2 } + 1 \right \} \right \}$$

En la expresión anterior se generalizó la sumatoria positiva correspondiente de los ceros y a la sumatoria negativa correspondiente a los polos , dejando en cada caso una expresión única con frecuencia de corte general ω c l . La variable independiente ω se emplea escala logarítmica, para que gráficamente se posibilite el desarrollo de aproximaciones asintóticas.

Década: factor de 10 entre dos frecuencias.

Octava: factor de 2 entre dos frecuencias.

Ejemplo: Entre las frecuencias ω 1 = 2 . 5 y ω 2 = 25 hay una década. Entre las frecuencias ω 1 = 2 y ω 2 = 8 hay dos octavas.

Note que una octava corresponde a log (2) = 0 , 30103 ... ≈ 0 , 3 décadas.

Por ello, 20 dB dec ≈ 6 dB oct

.
