## 4.4.3. Ampliación del rango de medida

Es posible en un instrumento electrónico ampliar el rango de una medición de tensión mediante resistores en serie que pueden ser incluidos de forma especial por las puntas de prueba . Ello genera una atenuación que simplemente se rige por el principio del divisor de tensión .

Para el caso de un rango de corriente se puede ampliar si se tiene una sensibilidad adecuada y se le agrega un resistor en paralelo. El factor de escala queda dado por la Ley de Ohm .

## Ejemplo:

Sea un multímetro digital de rangos desde 1mV a 1000V con una entrada de 10MΩ . Si se dispone de una punta de 20kV y un shunt de 10A , calcule las resistencias de las puntas y del shunt si se usan los rangos de 1000V y 100mV respectivamente.

## Respuesta:

La punta de tensión tendrá una resistencia de:

$$R _ { p u n t a } = 1 0 M \Omega \left ( \frac { 2 0 k V } { 1 k V } - 1 \right ) = 1 9 0 \ [ M \Omega ]$$

Y para el shunt se tiene:

$$R _ { s h } = \frac { 1 0 0 m V } { 1 0 A } = 1 0 \ [ m \Omega ]$$
