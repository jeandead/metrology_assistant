## 5.1.3. Trabajo con respuesta en frecuencia

Cuando el centro de interés es la respuesta en frecuencias de un sistema, se analiza de forma directa la función de transferencia H ( ω ) entre una variable de entrada respecto a una variable de salida. Luego, conocido y estudiado H ( ω ) , se procede a analizar el contenido de frecuencias de la señal de entrada para establecer cual será la señal de salida.

## Ejemplo:

De ahí se tiene que para frecuencias ω ≪ ω c el término 1 + ( ω/ω c ) 2 ≈ 1 , por lo que su logaritmo es nulo, quedando solamente el aporte de la ganancia K . Pero para frecuencias ω ≫ ω c el término 1 + ( ω/ω c ) 2 ≈ ( ω/ω c ) 2 , luego el término en decibel queda como -20 log 10 ( ω/ω c ) , lo cual por su signo negativo, a medida que la frecuencia crece se tiende a atenuar por completo la ganancia K .

Este procedimiento se conoce como análisis de diagrama asintótico o simplemente Diagrama de Bode , que para este caso corresponde solamente al diagrama de magnitud. Se aprecia que el parámetro fundamental de análisis es la frecuencia de corte ω c , que sumado al comportamiento de H ( ω ) = H ( ω/ω c ) permite analizar los rangos de frecuencias donde se producen las amplificaciones o atenuaciones en la ganancia de magnitud. Para este ejemplo en particular, se aprecia que H ( ω ) tiene un comportamiento de pasa bajos , es decir, que para frecuencias bajas específicamente menores que ω c , H ( ω ) amplifica de buena manera las entradas en frecuencia aplicadas ( V ( ω ) ) y para frecuencias mayores se produce una fuerte atenuación.
